# Integrate and extend

Choose the supported boundary that matches the work:

- [Send normalized records with Customer Push](customer-push/index.md)
- [Use supported APIs](api/index.md)
- [Configure provider webhooks](webhooks/index.md)
- [Build a supported integration](build-an-integration/index.md)

Customer Push is a versioned REST ingestion path. Provider webhooks are part of Full Chaos-managed synchronization. Analytics queries use the GraphQL API. Do not mix their tokens, endpoints, ownership, or retry behavior.
