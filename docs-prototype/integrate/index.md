# Integrate and extend

Send supported data, use APIs and webhooks, automate workflows, or build a supported integration.

## Common tasks

- Register a Customer Push source.
- Discover the supported schema.
- Submit records idempotently and handle errors.
- Authenticate and query the GraphQL API.
- Configure and verify webhooks.
- Build and test a provider integration.

## Reserved future area

Context Fabric is expected to need customer documentation, but no page or navigation node is published until supported tasks, terminology, ownership, and source-of-truth material are ready.
