# Build AI Impact dashboard

## Goal

Create a top-level AI Impact view that summarizes whether AI-assisted work is helping or creating hidden drag.

## Panels

- AI-assisted work share
- Agent-created work share
- Net AI delivery lift
- Review amplification
- Rework drag
- Revert and incident drag
- Test gap rate
- Best-fit automation opportunities

## UX principles

- Show trends, not individual rankings
- Make AI Operating Leverage decomposable
- Link every summary to evidence
- Keep unknown attribution visible
- Avoid productivity-score framing

## Acceptance criteria

- Dashboard works with live API data and demo data
- Summary cards drill into underlying PRs, repos, and teams
- AI Operating Leverage shows component breakdown
- Unknown attribution appears as its own bucket
- Page copy reinforces system-health framing, not surveillance
